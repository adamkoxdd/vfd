# Launcher

`vfdlauncher` is the VFD desktop command palette.

## Root categories

- Applications
- Files
- Themes
- Workspaces
- Desktop
- Power

## Navigation

- Type to filter the current category.
- Up/Down or Page Up/Page Down changes selection.
- Enter opens a category or executes an action.
- Escape returns to the root category; Escape at root closes.
- Backspace on an empty query also returns to root.

Power actions use i3-nagbar confirmation before logout, reboot, or shutdown.
