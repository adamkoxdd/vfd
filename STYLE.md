# VFD Interface Style

The default Lain theme uses a compact terminal-inspired visual language.

## Core roles

- Background: near-black
- Foreground: soft gray
- Accent: `#9b7fd4`
- Dim: muted blue-gray
- Selection: dark purple
- Alert: soft pink

## Interaction

- Keyboard navigation is first-class.
- Selection must remain visible without relying only on color.
- Animations are optional, brief, and never block input.
- Labels are concise and usually uppercase for launch surfaces.
- Borders and spacing should be consistent across applications.

## Layout

- Prefer dense lists over cards.
- Avoid decorative toolbars.
- Preserve a clear left, center, and right hierarchy where applicable.
- Use shared widget metrics instead of application-specific spacing constants.
